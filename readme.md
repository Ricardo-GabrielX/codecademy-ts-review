# TypeScript Refresher Lab 🧪

Repositório dedicado à revisão prática e aprofundamento dos fundamentos de TypeScript.

## 🛠️ Tecnologias
* TypeScript
* Node.js

## 📂 Sumário

* **`Types/RestaurantRecommender`**: Sistema de recomendação de restaurantes. Foco em lógica condicional, filtros de arrays e conversão segura de tipos de dados.
* **`Types/TypeMart`**: Implementação de um fluxo de compras em terminal. Foco em tipagem de variáveis, métodos de busca (`.find()`) e cálculos matemáticos.
* **`Types/*.ts`** *(arquivos avulsos)*: Exercícios rápidos para fixação de conceitos básicos (`any`, `typeAnnotations`, `typeShapes`).

## 🚀 Como executar
Para rodar qualquer um dos exercícios localmente:

```bash
# Navegue até a pasta do projeto desejado (exemplo: TypeMart)
cd Types/TypeMart

# Compile o código TypeScript para JavaScript (gerando o arquivo .js)
tsc

# Execute o arquivo gerado
node index.js
